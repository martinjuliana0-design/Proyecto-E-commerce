# Proyecto-E-commerce

## Descripción del proyecto:
Este proyecto analiza el desempeño de ventas y la gestión de inventarios de un e-commerce del sector alimenticio durante el período 2023–2025.
El objetivo principal es entender cómo evolucionan las ventas en el tiempo, identificar productos clave, evaluar si el inventario acompaña la demanda y detectar posibles ineficiencias en la gestión de stock.

-----

## Objetivos del proyecto:
El objetivo principal es entender cómo evolucionan las ventas en el tiempo, identificar productos clave en ingresos y unidades vendidas, evaluar si el inventario acompaña la demanda y detectar posibles ineficiencias en la gestión de stock.
Generar insights accionables para la toma de decisiones comerciales y operativas.

-----

## Metodología de Análisis:
*	El proyecto se desarrolló siguiendo una metodología estructurada de análisis de datos, orientada a la toma de decisiones de negocio.
* Comprensión del problema:Evaluar si la gestión de inventario acompaña la demanda real,Detectar posibles ineficiencias operativas (sobrestock o quiebres),Analizar el desempeño de los productos clave.
*	Exploración y preparación de los datos:Validación de consistencia entre totales (SQL, Excel y Power BI),Identificación y corrección de errores en: Unidades vendidas, estados de ordenes, movimientos de inventarios.
*	Cálculo de métricas y KPIs relacionados con ventas y movimientos de inventarios.
*	Visualización de datos a través de un dashboard interactivo
* Propuestas de mejora en la gestión de inventario,Recomendaciones para optimizar ventas y rotación de productos,Identificación de oportunidades para análisis futuros y automatización

----

## Hipótesis 
No todos los productos con más salidas de inventario generan mayor volumen de ventas, existiendo una diferencia entre las unidades vendidas registradas y las salidas de inventario asociadas a ventas, lo cual una mala gestión entre ventas e inventarios puede generar sobrestock o riesgo de quiebre.

## Principales conclusiones del análisis:
* Las ventas muestran patrones de estacionalidad, con picos en determinados meses.
* Algunos productos presentan mayores salidas de inventario que unidades vendidas ejecutadas, lo que sugiere ajustes operativos o diferencias en los registros.
* El inventario, en general, acompaña la demanda, pero existen productos con riesgo de sobrestock.
* Impacto económico potencial
* Optimización del inventario puede reducir costos de almacenamiento.
* Mejor alineación entre ventas e inventario disminuye pérdidas por quiebre o sobrestock.
* Identificación de productos clave permite focalizar estrategias comerciales y promociones.

## Evaluación bajo metodología SMART:
* S (Specific): analizar ventas e inventario del e-commerce.
* M (Measurable): KPIs claros y cuantificables.
* A (Achievable): análisis realizado con datos reales y herramientas estándar.
* R (Relevant): impacto directo en decisiones comerciales y operativas.
* T (Time-bound): período definido 2023–2025.

----

## Conclusion final:
El proyecto permitió analizar de forma integral las ventas y la gestión de inventarios, demostrando que la demanda del negocio no siempre se refleja de manera directa en los movimientos de stock.
Al diferenciar entre unidades vendidas y unidades vendidas ejecutadas, se identificó que solo una parte de las ventas impacta efectivamente en el inventario, dependiendo del estado de las órdenes. Esta distinción fue clave para comprender la relación real entre ventas y operación.
La comparación entre unidades vendidas ejecutadas y salidas de inventario por ventas evidenció desfasajes en algunos productos, lo que sugiere oportunidades de mejora en la planificación, rotación y control del stock.
En conjunto, el análisis muestra que integrar métricas de ventas con movimientos de inventario permite una toma de decisiones más precisa, ayudando a reducir riesgos de quiebre de stock, evitar sobreabastecimiento y mejorar la eficiencia operativa del negocio.
